# `magnitude()`

Returns the number of non-empty cells of a given quadrille row.

# Example

# Syntax

> `magnitude(row)`
 
# Parameters

| param    | description                                                                     |
|----------|---------------------------------------------------------------------------------|
| row      | Number: col number of the cell to be read [\[0..height\]]({{< ref "height" >}}) |

# `isString()`

Returns `true` if the cell found at `(row, col)` is a string and `false` otherwise.

# Example

# Syntax

> `isString(row, col)`

# Parameters

| param    | description                                                                     |
|----------|---------------------------------------------------------------------------------|
| row      | Number: col number of the cell to be read [\[0..height\]]({{< ref "height" >}}) |
| col      | Number: row number of the cell to be read [\[0..width\]]({{< ref "width" >}})   |
